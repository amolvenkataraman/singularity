# Singularity
A command-line script that can download content, announcements, and more information from Google Classroom or D2L Brightspace to a local directory in standardized file formats.

## Why does it exist?
Towards the end of my final year in high school, many of my teachers suggested that I keep a copy of all the notes and slides that they created, as it can be a useful reference during post secondary education. But since our online accounts would be deleted when we graduate, I could not rely on accessing it directly through the learning platforms. So, I made this program to download a local copy of all the data from online classroom websites so that I could keep a copy of learning resources even after you graduate.

## What does it do?
Here are some things that Singularity can do:
- Download all content, announcements, and other data from Google Classroom and D2L Brigtspace.
- Download almost all possible file types stored in a myriad of different sources, and provide a list of any files that could not be downloaded.
- Run in the background without any human intervention needed.
- Automatically pick up where it left of if interrupted.
